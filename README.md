## 📋 Sobre o projeto
<p>O e-Ticket é uma aplicação web interativa que simula a compra de ingressos para eventos em diferentes setores (Pista, Cadeira Superior e Cadeira Inferior).
  O projeto foca na experiência do usuário e na integridade dos dados, garantindo que as compras só ocorram se houver disponibilidade no estoque.</p>


## ✦ Principais funcionalidades:

<p>➥ Seleção de Setores: Interface intuitiva para escolher entre diferentes categorias de assentos.</p>
<p>➥ Validação de Estoque: O sistema verifica em tempo real se a quantidade solicitada é superior à disponível.</p>
<p>➥ Atualização Dinâmica do DOM: Ao realizar uma compra bem-sucedida, a quantidade disponível na tela é atualizada instantaneamente sem recarregar a página.</p>
<p>➥ Feedback ao Usuário: Alertas personalizados para confirmar o sucesso da transação ou informar a indisponibilidade de ingressos.</p>
